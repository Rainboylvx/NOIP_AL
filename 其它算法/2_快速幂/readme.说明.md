###快速幂，快速幂取模(余)

###原理
如果我们求 a^11 =a^b1011
b1011 =2^3+2^1+2^0
也就是 
b1011 = 1*2^3 + 0*2^2+1*2^1+1*2^0

所以 
a^11 = a^2^3 * a^2^1*a^2^0
注意每一位的变化
