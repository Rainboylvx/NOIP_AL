TarjanDFS求强连通分量

前面

写出要求强连通分量图中的点DFS访问顺序,找规律.

原理

例:原图

DFS形成的树:

1.在DFS中,同一强连通分量内的所有顶点都在同一棵深度优先树搜索树中,也就是说强连通分量一定是DFS树的子树

2.如果我们从DFS树的最低层每个强连通分量的根以下的点都输出,那就可以一个个输出强连通分量了

3.如何确定DFS树中的某个点是一个强联通分量的根?

有这样的一些性质:

  • 因为强连通分量里的点都可以互相访问
  • 在DFS的过程中,如果某个点A能访问比自己访问时间更早的点B,那A和B可以互相访问
  • 如果点A不能找到一个比自己访问时间更早的点B,那这个点一定当前强联通分量的根

反证法证明第三点:

如果点A不是强联通分量的根,点A一定属于某个强连通分量,那一定有某个回路可以让A回到祖宗(根),那A一定可以访问到比较自己时间更少的点

拿出强连通分量:

如果当前结点A为一个强联通分量的要,那么它的强连通分量一定是以A为根的子树.

  • 在DFS的过程中维护一个堆栈,每次访问一个新的结点,就压入堆栈
  • 点A是最先被压入堆栈的
  • 点A以后被压入堆栈的点(仍在堆栈中),都属于这个强联通分量

算法过程

我们维护两个为群组,index,low,一个堆栈stack

  • index[i]表示点i开始访问的时间
  • low[i]表示点i能访问访问过点中的最小的时间
  • stack按访问顺序压入点,DFS回溯到强连通分量的根的时候,弹出根后面的点

具体代码

算法实现过程:

  • 初始化index[x]和low[x]
  • 对x的所有临接点'v':
    • 如果没有被访问过,则访问v,同时维护low[x]
    • 如果没有被访问过,但没有被删除,就维护low[x]
  • 如果index[x] == low[x],输出

/* -----------
 *  原理,:一个分量里的点都可以互相访问
 * 算法实现:
 *      1.初始化 dfn[i], low[i]
 *      2.对i所有相临的结点j
 *          1.if !vis[j] dfs[j]
 *          2.if vis[j] low[i] =dfn[j]
 *      3.if dfn[i] = low[i]
 * */


#include <cstdio>
#include <cstring>


#define maxn 1000 //最多的点
bool instack[maxn] ={0}; // =false

int low[maxn];
int dfn[maxn] ={0};

int color[maxn]; // color[i] 点i的强连通分量

int stack[maxn],top; // 自己维护的stack ,可以用 STL stack
int cnt =0,index=0;

//存图
int first[maxn];
int u[maxn],v[maxn],next[maxn];

void tarjan(int x){
    int j;
    dfn[x] = low[x] = ++index;
    instack[x] =true; //在栈内,可以用来判断这个是不是已经被访问过
    stack[++top] = x;
    int e = first[x];
    for(; e != -1; e = next[e]){
        j = v[e];
        if( !dfn[j] ) // dnf[j] == 0 表示 这个点没有被访问过
        {
            tarjan(j);
            if(low[j] < low[x])
                low[x] = low[j];
        } 
        else if(instack[j] && low[j] < low[x]) //更新,访问过没有被删除
            low[x] = low[j];
    }
    if(dfn[x] == low[x])
        {
            cnt++;
            do{
                j=stack[top--];
                instack[j] = false;
                color[j] = cnt;
            }
            while(j !=x);
        }
}

int main(){
    freopen("1.in","r",stdin);
    top = 0;
    int n,i,k;
    int T;
    memset(first,-1,sizeof(first));
    scanf("%d%d",&n,&T);
    //读取图 这里用边集数组
    for(i=1;i<=n;i++){
        scanf("%d%d",&u[i],&v[i]);
        next[i] = first[u[i]];
        first[u[i]] = i;
    }
    for(i=1;i<=T;i++){ // 遍历所有点
        if(!dfn[i]) // 没有被访问过
            tarjan(i);
    }
    printf("%d\n",cnt);
    return 0;
}

如何证明TarjanDFS是对的?

  1. low() 一个操作一定是对的

在DFS中,同一强连通分量内的所有顶点都在同一棵深度优先树搜索树中,也就是说强连通分量一定是DFS树的子树

这个子树的元素在栈中一定是相临的,且root在子树所有元素的下面 dfn用来记录

强连通分量是由若干个环组成的。所以，当有环形成时（也就是搜索的下一个点已在栈中），我们将这一条路径的low值统一，即这条路径上的点属于同一个强连通分量。

如果dfn=low,那这个点一定是root,输出

按我们的定义,DFS子树上的low 值应该一样,这是一种理解状态,但是算法的过程中,应该能实际达到一样.

猜想:当点i low[i]=dfn[i] 时 i为root点,输出 stack.top -> stack.i的元素,

如果一个点i已经 遍历完它的所有相临点,有如果有一个点j,且点dfn[j] < dfn[i]那low[i] 一定会变得比dfn[i] 小(但是不是理论值,dfn[root])

low[i] 应该是代表i相连的最早的点

/-------------------/ DFS 中一个强连通分量 里的点 可以相互访问,所以一定在一棵DFS树中,又因为一个DFS树可能有多个强连通分量,所以一定是DFS对的子树
